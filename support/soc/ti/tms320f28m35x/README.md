# TI TMS320F28M35x SoC Entry

This entry tracks the Concerto dual-core lane in a Zephyr-style SoC location.
The OpenOCD runtime target remains `tcl/target/ti/tms320f28m35x.cfg`.

Current status is experimental: C28x target creation loads locally, while real
ICEPick secondary-TAP discovery, debug operations, and flash are still gated on
hardware validation.
