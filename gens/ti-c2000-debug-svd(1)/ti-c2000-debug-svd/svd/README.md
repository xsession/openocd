# Generated output

This directory receives generated SVD files:

- `mspm0c1103.svd` - Cortex-Debug compatible Cortex-M0+ view
- `tms320f28m35x_m3.svd` - Cortex-Debug compatible Cortex-M3 subsystem view
- `tms320f28m35x_c28x.svd` - C28x peripheral-viewer view
- `tms320f28069.svd` - C28x peripheral-viewer view
- `tms320f280049.svd` - C28x peripheral-viewer view

The files are generated from pinned TI CMSIS packs or the locally installed CCS
target database rather than being hand-maintained. An SVD file does not supply a
GDB server, flash algorithm, or CPU debug implementation.
